# Training an LLM in Swift, Part 1: Taking matrix mult from Gflop/s to Tflop/s

> **来源**: [hacker_news](https://cocoawithlove.com) | **热度**: 230 | **重要性**: 值得了解（6/10）

## 摘要

本文记录了用 Swift 从零构建 LLM 训练基础设施的第一阶段探索，核心成果是将矩阵乘法性能从 Gflop/s 级提升到 Tflop/s 级，跨越约三个量级。这一工作对于 Apple Silicon 生态下的本地 LLM 训练具有直接参考价值，也佐证了 Swift 在高性能计算领域的可行性。随着 Apple 持续增强 Metal 和 MLX 框架的能力，Swift 有望成为 Apple 平台 ML 工程的重要语言选项。该系列文章的技术深度和工程细节对 ML 系统工程师有较高学习价值。

## 核心亮点

- 矩阵乘法从 Gflop/s 到 Tflop/s 的三个量级优化，展示了 Swift 在底层 ML 计算的真实潜力
- 为 Apple Silicon 生态下的 LLM 训练基础设施探索开辟了新路径
