# Tuning Deep Agents to Work Well with Different Models

> **来源**: [langchain_blog](https://www.langchain.com/blog/tuning-deep-agents-different-models) | **热度**: 18 | **重要性**: 值得了解（6/10）

## 摘要

LangChain Deep Agents 的模型适配优化研究揭示了一个重要工程事实：即使是同类型的 Agent 任务，针对不同模型厂商的专属配置（Prompt、工具设计、中间件策略）可以带来 10-20 分的显著性能提升。这意味着"模型无关"的 Agent 框架并不等于"配置无关"——达到最优性能需要针对具体模型进行专项调优。对于 harness9 等支持多模型 Provider 的框架，这一研究提示了在模型切换时需同步调整系统提示和工具定义的重要性。tau2-bench 上的量化结果也为评估 Agent 框架的模型适配工程投入价值提供了参考基准。

## 核心亮点

- 不同模型厂商的 Agent 最优配置差异显著，专属配置文件可带来 10-20 分的基准测试提升
- Prompt、工具和中间件三层适配策略为模型无关框架的性能调优提供了方法论参考
- tau2-bench 量化数据支撑，为 Agent 框架的模型适配工程提供了可参照的性能提升范围
