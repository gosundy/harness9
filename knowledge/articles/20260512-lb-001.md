# Open Models have crossed a threshold

> **来源**: [langchain_blog](https://www.langchain.com/blog/open-models-have-crossed-a-threshold) | **热度**: 22 | **重要性**: 改变格局（9/10）

## 摘要

LangChain 对 Agent 场景模型选型的这篇研究标志着一个重要转折点：以 GLM-5 和 MiniMax M2.7 为代表的开源模型在工具调用、文件操作、指令遵循等 Agent 核心能力上已达到与 GPT-4/Claude 级别闭源模型相当的水平。结合其显著更低的成本和延迟优势，这意味着企业构建 Agent 系统时的供应商锁定风险大幅降低，技术选型的自由度大幅提升。这一拐点的到来将深刻影响 Agent Harness 框架（如 harness9）的 Provider 层设计，多模型灵活切换的架构价值将愈发凸显，单一闭源模型依赖的风险也将推动更多团队转向模型无关的开放架构。

## 核心亮点

- 开源模型在文件操作、工具调用、指令遵循等 Agent 核心任务上首次达到闭源前沿模型水平
- 成本和延迟大幅优于闭源模型，为大规模 Agent 部署提供了经济可行的新选项
- GLM-5 和 MiniMax M2.7 成为标志性的阶段性里程碑，Agent 场景开源替代方案成熟
- 这一拐点将推动企业 Agent 选型从闭源优先转向开源优先评估
