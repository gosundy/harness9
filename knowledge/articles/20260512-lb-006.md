# Open SWE: An Open-Source Framework for Internal Coding Agents

> **来源**: [langchain_blog](https://www.langchain.com/blog/open-swe-an-open-source-framework-for-internal-coding-agents) | **热度**: 22 | **重要性**: 直接有帮助（7/10）

## 摘要

Open SWE 是 LangChain 生态中首个面向企业内部编码 Agent 场景的完整开源框架，基于 Deep Agents 和 LangGraph 构建，覆盖了从 GitHub Issue 规划到最终 PR 提交的完整自动化链路。其异步云端执行架构解决了内联 Agent 阻塞主流程的痛点，是当前最接近生产可用的开源编码 Agent 参考实现。对于希望构建内部自主编码 Agent 的企业技术团队，Open SWE 提供了一个起点，无需从零设计架构。同时，其 PR 自动提交能力也印证了编码 Agent 正在从"辅助编码"向"自主完成任务"的关键能力跃迁。

## 核心亮点

- 完整实现了编码 Agent 全流程：GitHub 任务规划→编码→测试→PR 提交，端到端自动化
- 开源发布为企业内部构建自主编码 Agent 提供了可直接参考的架构蓝本
- 异步执行架构支持云端自主运行，大幅提升大规模任务的处理效率
- 基于 Deep Agents + LangGraph 的技术选型代表了当前 LangChain 生态的最佳实践
