## 摘要

Nous Research 推出的 hermes-agent 代表了 AI Agent 从"工具调用"向"自主技能积累"范式演进的重要里程碑。其核心创新在于 Agent 可从真实执行经验中动态创建可复用技能，并通过跨会话记忆持久化机制实现真正意义上的持续学习。该项目以 14.5 万 Star 的高热度位列 GitHub 趋势榜首，广泛的模型提供商兼容性（200+）和多平台接入能力使其具备极强的生产落地价值。对于正在构建 Agent Harness 的团队而言，其 RL 环境集成设计和并行委派机制提供了重要的架构参考。

## 核心亮点

- 自进化架构：Agent 可从执行经验中自主创建并积累技能，无需人工定义工具
- 跨会话持久化记忆，彻底解决长期任务中上下文丢失的问题
- 兼容 200+ 模型提供商，多平台接入（Telegram/Discord/Slack/CLI）极大降低集成成本
- 支持并行 Agent 委派，具备 RL 环境集成能力，为自主 Agent 研究提供重要基础设施
- 热度高达 14.5 万 Star，是当前 GitHub Trending 最受关注的 Agent 开源项目
